# Notebook 2

This template should help get you started developing with Vue 3 in Vite.

## Backend Setup

### Look at backend readme to get started

### Start Backend
```
python3 manage.py runserver
```

## Frontend Setup

### Install Dependencies
```
npm install
```

### Start Project for Development
```
npm run dev
```
